# Setup for the project 

## Moxa 

The moxa we are using is a NPort 5650-8-DT. 
The following ports are configured:

IP setup (static):

- IP: 192.168.1.2 
- Subnet mask: 255.255.255.0

- Port 1: 
  - Agilent 4UHV Ion pump controller 
  - Serial setup: 38_400, 8, N, 1
  - Mode: TCP Server
  - IP Port: 4001
